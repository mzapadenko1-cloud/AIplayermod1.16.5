# 🤖 AI Player Mod v5.0
### Minecraft 1.16.5 Fabric

ИИ-игрок на базе Gemini AI — живёт рядом с тобой, развивается, строит, думает, видит сны.

---

## ⚡ Быстрая установка (без компиляции)

### Шаг 1 — Скачать готовый JAR
1. Открой вкладку **Actions** на этой странице GitHub
2. Нажми на последний успешный билд (зелёная галочка ✅)
3. Внизу страницы найди **Artifacts** → скачай `aiplayermod-jar.zip`
4. Распакуй — внутри будет `aiplayermod-5.0.0.jar`

### Шаг 2 — Установить Fabric
- Скачай [TLauncher](https://tlauncher.org) если ещё нет
- При создании версии выбери **Fabric 1.16.5**

### Шаг 3 — Установить моды
Скопируй в папку `mods/` (обычно `%appdata%\.minecraft\mods\`):
- `aiplayermod-5.0.0.jar` — этот мод
- [Fabric API для 1.16.5](https://modrinth.com/mod/fabric-api/versions?g=1.16.5) — обязательно!
- *(опционально)* [Baritone Fabric](https://github.com/cabaletta/baritone/releases) — для реальной навигации

### Шаг 4 — Получить API ключ
1. Открой [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
2. Нажми **Create API Key** — это бесплатно
3. Скопируй ключ

### Шаг 5 — Запустить
1. Зайди в мир
2. Нажми **клавишу I** → вставь API ключ → нажми Сохранить
3. Введи `/aiplayer start <твой_ник>`
4. Наблюдай 🎉

## 🎮 Команды

| Команда | Что делает |
|---------|-----------|
| `/aiplayer start <ник>` | Запустить ИИ |
| `/aiplayer stop` | Остановить |
| `/aiplayer status` | Состояние |
| `/aiplayer memory` | История действий |
| `/ai ...` | Короткий алиас |

## 🧠 Что умеет ИИ

- **Выживание** — сам добывает ресурсы, крафтит инструменты, строит укрытие
- **Эмоции** — 9 состояний: страх, восторг, скука, мания, меланхолия и др.
- **МАНИЯ** — после долгой скуки строит Кёльнский собор из верстаков
- **Долгосрочные цели** — планирует на несколько дней вперёд
- **Постоянная память** — помнит всё между перезапусками
- **Сны** — видит сны когда спит, рассказывает утром
- **Внутренний монолог** — думает вслух (читается в GUI)
- **Изучение предметов** — сам разбирается с новыми модами
- **Любимые места** — запоминает и возвращается к ним
- **Реакция на смерть** — экзистенциальный шок, травма, страхи
- **Погода** — дождь делает меланхоличным, гроза пугает
- **Усталость** — устаёт и берёт осознанный отдых
- **Диалоги** — философствует, задаёт вопросы, реагирует на ответы
- **Отношения** — помнит как ты с ним общаешься
